# 1. Prediction Circle

Remove the hardcoded ₹2730 and replace it with:

```jsx
₹{predictAmount}
Predicted Bill
```

The circular progress should NOT depend on percentage change.

Instead, use a fixed maximum bill value:

```js
const MAX_BILL = 7000;

const progress = Math.min(
  (predictAmount / MAX_BILL) * 100,
  100
);
```

Examples:

* ₹1750 = 25%
* ₹3500 = 50%
* ₹5250 = 75%
* ₹7000 = 100%
* Any value above ₹7000 remains capped at 100%

Create an animated conic-gradient ring:

```css
background:
conic-gradient(
  #6D4AFF 0deg,
  #7B61FF calc(var(--progress) * 3.6deg),
  #E5E7EB calc(var(--progress) * 3.6deg)
);
```

Ring requirements:

* thickness around 18px
* soft shadow
* smooth 1.5s animation
* rounded appearance
* light gray remaining section

Center:

* large bold amount
* "Predicted Bill" below
